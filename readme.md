# Starting the project

To start the setup, execute the commands in the sequence below

```
git clone https://github.com/omateusventura/setup-node.git

```

```
npm install
```

```
npm run dev
```

# Currently, the setup has these packages

- Express
- Typescript
- Jest and Supertest
- Axios
- Dot Env
- PGPromise
- UUID
- Class Validator
- Class Transformar
